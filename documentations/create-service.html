--- layout: post title: How to create a service in Linux link: create-service category: documentations type: documentations ---

This can be used with How to create a RPM package in Linux

  1. Create a service file for execuatble file:
    Example: luchecker.service
    [Unit]
    Description=linux updates checking service
    After=network.target
    StartLimitBurst=5
    StartLimitIntervalSec=10

    [Service]
    Type=simple
    Restart=always
    RestartSec=1
    User=root
    ExecStart=/usr/bin/linux-updates-checker

    [Install]
    WantedBy=multi-user.target
  2. Put it to /lib/systemd/system
  3. Run and enable the service (when the service is enabled, it will start with the system):
    $ systemctl start luchecker.service
    $ systemctl enable luchecker.service
