Check quota:

$ quota -v

Report the space left on the file system:

$ df -T -h

To display the largest folders/files including the sub-directories, run:

du -hs * | sort -rh | head -5
du -Sh | sort -rh | head -5

Mount NFS:

mount -t nfs -o proto=tcp,port=2049 10.37.129.2:/ /mnt/sites

Docker run nginx (only remind for me):

docker run -d -p 80:80 -p 443:443 --name=nginx -v /mnt/sites:/usr/share/nginx/html nginx:v2

Find files: $ find . -name testfile.txt
