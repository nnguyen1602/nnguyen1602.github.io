--- layout: post title: How to Reset Root Password of GitLab link: reset-gitlab-password category: documentations type: documentations ---

Note: make sure gitlab and gitlab console are running

  1. Start the console with this command:
    $ gitlab-rails console production
  2. Choose user 1 (root):
    $ user = User.where(id: 1).first
    or with email:
    user = User.find_by(email: 'admin@local.host')
  3. Change password:
    $ user.password = 'secret_pass'
    $ user.password_confirmation = 'secret_pass'
  4. Save:
    $ user.save!
