--- layout: post title: How to run FTP server in Linux link: install-ftp-server category: documentations type: documentations ---

Note: this tutorial is only for CentOS 7 / RHEL 7

  1. Install vsftpd (server) and ftp (client):
    $ yum install vsftpd ftp -y
  2. Change in the config file: "/etc/vsftpd/vsftpd.conf"
    anonymous_enable=NO
    local_enable=YES
    chroot_local_user=YES
  3. Restart the service:
    $ systemctl restart vsftpd
